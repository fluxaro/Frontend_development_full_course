HTML Images — Complete Demo

This page demonstrates every way to use images in HTML correctly.

1. Basic Image with Alt Text

Every image needs an alt attribute, width, and height.

2. Lazy Loading

Add loading="lazy" to images below the fold. They only download when the user scrolls to them.

3. Figure with Caption

Use <figure> and <figcaption> to associate a caption with an image.

4. Picture Element — Format Fallback

Serve WebP to modern browsers, JPEG to older ones. The browser picks the first format it supports.

5. Picture Element — Responsive (Art Direction)

Show a different crop of the image on mobile vs desktop. Resize your browser to see the difference.

6. Decorative Image — Empty Alt

Purely decorative images get alt="" so screen readers skip them.

7. Image as a Link

When an image is a link, the alt text describes the destination, not the image.

Complete Head Section Example

This page demonstrates all the important tags that belong inside the <head> element.

Right-click this page and choose "View Page Source" to see the full head section.

What Goes in the Head?

  • meta charset — sets the character encoding
  • meta viewport — controls mobile display
  • title — the page title in the browser tab
  • meta description — shown in search results
  • meta og:* — Open Graph tags for social sharing
  • link rel="icon" — the favicon
  • link rel="stylesheet" — links to CSS files
  • link rel="canonical" — preferred URL for SEO

What Does NOT Go in the Head?

  • Visible content (text, images, headings)
  • Navigation menus
  • Paragraphs or lists
  • Anything the user sees on the page

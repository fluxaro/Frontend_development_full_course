Paragraphs and Line Breaks

This is a paragraph. Paragraphs are the most common text element. They add space above and below automatically.

This is a second paragraph. Notice the space between them.

This line has a break
right here using the br tag.

Emphasis and Importance

Use strong for important text. It is bold and carries semantic meaning.

Use em for emphasized text. It is italic and carries semantic meaning.

Use b for bold text without semantic importance.

Use i for italic text without semantic importance (like titles or technical terms).

Highlighting and Edits

Use mark to highlight text, like a yellow highlighter.

Use del for deleted text (shows as strikethrough).

Use ins for inserted text (shows as underlined).

Price was $99, now $49!

Superscript and Subscript

Water is H2O — the 2 is a subscript.

Einstein's equation: E = mc2 — the 2 is a superscript.

The footnote1 is marked with a superscript number.

Quotations

Use blockquote for long quotes from another source:

The only way to do great work is to love what you do. If you have not found it yet, keep looking. Do not settle.

— Steve Jobs

Use q for short inline quotes: To be or not to be, that is the question.

Code and Preformatted Text

Use code for inline code like console.log("Hello").

Use pre for blocks of code that preserve whitespace and line breaks:

function greet(name) {
  return "Hello, " + name + "!";
}

console.log(greet("World"));

Use kbd for keyboard input: Press Ctrl + C to copy.

Use samp for sample output: Error: File not found

Abbreviations and Definitions

HTML is the standard language for web pages.

CSS is used to style HTML elements.

The term semantic HTML means using HTML elements for their intended purpose.

Small and Address

Use small for fine print, disclaimers, or copyright notices.

Copyright 2025. All rights reserved.

alex@example.com

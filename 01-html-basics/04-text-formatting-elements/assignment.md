# Assignment — Tech News Article

## What You Are Building

Create a "Tech News Article" HTML page using at least 8 different text formatting elements. The article should read like something you would find on a real tech news website.

No CSS. No JavaScript. Pure HTML structure only.

---

## Requirements

### Must Use (at least 8 of these elements)

- `<h1>` — article title (exactly 1)
- `<h2>` — section headings (at least 2)
- `<h3>` — subsection heading (at least 1)
- `<strong>` — important text (at least 2 uses)
- `<em>` — emphasized text (at least 2 uses)
- `<mark>` — highlighted key point (at least 1)
- `<del>` and `<ins>` — a correction or update (at least 1 pair)
- `<blockquote>` — a quote from a source (at least 1)
- `<code>` — inline code or technical notation (at least 1)
- `<time>` with `datetime` attribute — publication date (at least 1)
- `<abbr>` with `title` attribute — an abbreviation (at least 1)
- `<small>` — fine print or secondary info (at least 1)
- `<sup>` or `<sub>` — superscript or subscript (at least 1)

### Content Requirements

- The article must be about a real or fictional tech topic
- At least 300 words of content
- At least one quote from a named source using `<blockquote>`
- At least one piece of code or technical notation using `<code>`
- A publication date using `<time datetime="...">`
- An author byline

### Document Structure

- Proper `<!DOCTYPE html>`, `<html lang>`, `<head>`, `<body>`
- Meaningful `<title>` and `<meta name="description">`
- Use `<article>` to wrap the article content
- Use `<header>` for the article title and byline
- Use `<footer>` for the copyright or author info

---

## Topic Ideas

- "OpenAI Releases New Model: What You Need to Know"
- "Why TypeScript is Taking Over JavaScript Development"
- "The Rise of AI Coding Assistants: A Developer's Perspective"
- "5 CSS Features That Changed Web Development in 2025"
- "How Quantum Computing Will Change Encryption"
- "The History of the World Wide Web: From 1991 to Today"

---

## What Good Looks Like

- Every element is used for its correct semantic purpose (not just for visual effect)
- `<strong>` is used for genuinely important information, not just bold text
- `<em>` is used for genuine emphasis, not just italic text
- `<blockquote>` contains an actual quote from a named source
- `<time>` has a valid `datetime` attribute (e.g., `datetime="2025-01-15"`)
- `<abbr>` has a `title` attribute with the full expansion
- The article reads naturally and makes sense

- Use `<figure>` and `<figcaption>` around a description of an image
- Add `<kbd>` for keyboard shortcuts mentioned in the article

HTML Lists — All Three Types

HTML has three types of lists. Each one is used for a different purpose.

1. Unordered List — ul

Use when the order does not matter. Items appear with bullet points.

Example: Shopping List

  • Milk
  • Eggs
  • Bread
  • Butter
  • Coffee

Example: Skills List

  • HTML
  • CSS
  • JavaScript
  • React
  • Git

2. Ordered List — ol

Use when the order matters. Items appear numbered.

Example: How to Make Tea

  1. Boil water in a kettle
  2. Place a tea bag in your mug
  3. Pour the boiling water over the tea bag
  4. Wait 3–5 minutes for the tea to steep
  5. Remove the tea bag
  6. Add milk or sugar if desired
  7. Enjoy your tea

Example: Top 5 Programming Languages (with start attribute)

  1. JavaScript
  2. Python
  3. TypeScript
  4. Java
  5. C#

Example: Countdown (reversed attribute)

  1. One
  2. Two
  3. Three
  4. Four
  5. Five

Example: Using Letters (type="A")

  1. Option A
  2. Option B
  3. Option C

3. Description List — dl

Use for term-definition pairs. Like a glossary or dictionary.

Example: Web Development Glossary

HTML
HyperText Markup Language. The structure of web pages.
CSS
Cascading Style Sheets. The styling of web pages.
JavaScript
A programming language that makes web pages interactive.
API
Application Programming Interface. A way for programs to communicate with each other.
DOM
Document Object Model. The browser's representation of an HTML page as a tree of objects.

4. Nested Lists

You can put a list inside a list item to create sub-items.

Navigation menus are built with unordered lists. CSS makes them look like a menu bar.

External Links

External links point to other websites. Use target="_blank" to open in a new tab.

  • Google (same tab)
  • GitHub (new tab)
  • MDN Web Docs (new tab)

Note: Always add rel="noopener noreferrer" when using target="_blank" for security.

Internal Links (Relative Paths)

Internal links point to other pages within the same website.

  • Home Page
  • About Page
  • Contact Page
  • Blog Post (in subfolder)
  • Parent folder index

Anchor Links (Jump to Section)

Anchor links jump to a specific section on the same page using the element's id.

Special Links

Email Link

Opens the user's email client with a new message:

Send me an email

With subject and body pre-filled:

Email with subject

Phone Link

On mobile, this opens the phone dialer:

Call +1 (555) 123-4567

Download Link

The download attribute makes the browser download the file instead of opening it:

Download My Resume (PDF)

You can also specify a filename:

Download Resume with custom filename

Link Attributes Reference

Common Link Attributes
Attribute Value Purpose
href URL or path The destination of the link
target _blank, _self Where to open the link
rel noopener noreferrer Security for external links
download filename (optional) Download the file instead of opening it
title Text Tooltip shown on hover

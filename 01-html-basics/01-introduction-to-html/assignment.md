# Assignment — About Me Page

## What You Are Building

Create a personal "About Me" HTML page from scratch. This is your first real HTML project — make it yours.

No CSS. No JavaScript. Pure HTML structure only.

---

## Requirements

Your page must include all of the following:

### 1. Proper HTML Structure
- `<!DOCTYPE html>` as the very first line
- `<html lang="en">` as the root element
- `<head>` with `<meta charset="UTF-8">` and a `<title>`
- `<body>` containing all visible content

### 2. Your Name as an `<h1>`
Use your real name (or a nickname if you prefer) as the main heading.

### 3. A Short Bio Paragraph
Write 2–4 sentences about yourself. Where are you from? What do you do? Why are you learning web development? Write it in your own words.

### 4. A List of 3 Hobbies
Use an unordered list (`<ul>`) with at least 3 list items (`<li>`).

### 5. A Link to Your Favorite Website
Include an anchor tag (`<a>`) that links to a real website. Add `target="_blank"` so it opens in a new tab.

---

## What Good Looks Like

- The page opens in a browser with no errors
- The heading is your actual name
- The bio paragraph is personal and written in your own words
- The list has real hobbies, not placeholder text
- The link works and opens in a new tab
- The HTML is properly indented and easy to read

---

## Bonus Challenges

- Add a second section with your favorite books, movies, or music
- Add an `<h2>` for "My Goals" with a paragraph about what you want to build
- Add more than one link
- Add a horizontal rule (`<hr>`) between sections
- Add a second list using an ordered list (`<ol>`) for your top 5 favorite things

---

## Submission

Save your file as `about-me.html` and submit via GitHub or file upload.

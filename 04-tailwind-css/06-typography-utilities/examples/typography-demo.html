Line Height

leading-tight

This text has tight line height. Good for headings but hard to read in paragraphs.

leading-relaxed

This text has relaxed line height. Perfect for body text and comfortable reading.

leading-loose

This text has loose line height. Very airy, good for short UI labels.

Truncation & Line Clamp

truncate

This is a very long text that will be cut off with an ellipsis when it overflows the container width.

line-clamp-2

This paragraph will be clamped to exactly two lines. Any content beyond the second line will be hidden and an ellipsis will appear at the end of the visible text.

line-clamp-3

This paragraph will be clamped to exactly three lines. Any content beyond the third line will be hidden. This is very useful for card components where you want consistent heights across all cards in a grid layout.

Building Better Typography with Tailwind

May 11, 2026 · 5 min read

Good typography is the foundation of readable, professional interfaces. Tailwind gives you a complete set of utilities to control every aspect of text rendering.

From font size and weight to line height and letter spacing, each utility maps directly to a CSS property — making it easy to build consistent, beautiful text without writing a single line of custom CSS.

The Complete Guide to Typography in Modern Web Design

Typography is the backbone of every great design. Learn how to use font scales, line heights, and letter spacing to create interfaces that are both beautiful and readable.

AK

Alex Kim

Senior Frontend Engineer · 8 min read · June 12, 2025

Good typography is invisible. When it works, readers don't notice it — they just absorb the content effortlessly. When it fails, everything feels off: text is too small, lines are too long, or the hierarchy is unclear. In this guide, we'll break down the core principles and show you exactly how to apply them using Tailwind CSS.

Understanding the Type Scale

A type scale is a set of font sizes that work harmoniously together. Tailwind's default scale follows a modular ratio, meaning each step is proportionally larger than the last. This creates visual consistency across your entire interface.

Tailwind Type Scale

text-xs    →  0.75rem  (12px)
text-sm    →  0.875rem (14px)
text-base  →  1rem     (16px)  ← body default
text-lg    →  1.125rem (18px)
text-xl    →  1.25rem  (20px)
text-2xl   →  1.5rem   (24px)
text-3xl   →  1.875rem (30px)
text-4xl   →  2.25rem  (36px)
text-5xl   →  3rem     (48px)
text-6xl   →  3.75rem  (60px)

Line Height: The Invisible Grid

Line height (or leading) controls the vertical space between lines of text. It's one of the most impactful — and most overlooked — typographic decisions you'll make. Too tight and text feels claustrophobic. Too loose and the eye struggles to track from line to line.

leading-tight

Use for headings and display text where you want a compact, punchy feel.

leading-relaxed ✓

The sweet spot for body copy. Comfortable to read, especially in longer paragraphs.

leading-loose

Very airy. Good for short captions or UI labels, not long paragraphs.

Letter Spacing and Hierarchy

Letter spacing (tracking) is a subtle but powerful tool. Wide tracking on small, uppercase text creates a polished label effect. Tight tracking on large display text makes headings feel more impactful and modern.

When to use each tracking value

  • →
    tracking-tighter — Large display headings (text-5xl and above)
  • →
    tracking-tight — Section headings (text-2xl to text-4xl)
  • →
    tracking-normal — Body text and most UI elements
  • →
    tracking-wide — Subheadings and captions
  • →
    tracking-widest — Labels, tags, and category badges (always uppercase)

Putting It All Together

Great typography comes down to a few consistent decisions: a clear size hierarchy, appropriate line heights for each context, and restrained use of font weights to guide the eye. With Tailwind, every one of these decisions is a single class away.

The best way to internalize these principles is to build. Take any design you admire, try to recreate its typography using only Tailwind utilities, and pay attention to which classes you reach for most often. You'll develop an intuition for it faster than you think.

# Assignment — Typography Utilities

## Goal

Build a styled article page using only Tailwind typography utilities. No custom CSS. The page should demonstrate mastery of font size, weight, color, spacing, and text transforms.

## Requirements

- [ ] Create `article-page.html` with the Tailwind CDN in the `<head>`
- [ ] Include a page header with a large title (`text-4xl` or bigger), a subtitle, and a category label using `uppercase` and `tracking-widest`
- [ ] Write at least 3 paragraphs of body text using `leading-relaxed` for comfortable reading
- [ ] Add a blockquote or pull-quote styled with a larger font size, different color, and `italic`
- [ ] Include a section with a list where each item uses a different font weight to show the scale
- [ ] Add at least one `truncate` example — a card or sidebar item where long text is cut off
- [ ] Use `font-mono` somewhere (code snippet, stat, or label)
- [ ] Use at least 3 different text colors from the Tailwind palette

## What Good Looks Like

- The page has clear visual hierarchy — headings are obviously larger and bolder than body text
- Body text is comfortable to read (not too tight, not too loose)
- The category label looks like a label — small, uppercase, wide tracking
- The truncated element actually truncates (has a width constraint)
- No custom CSS in a `<style>` block — all styling via Tailwind classes

## Submission

Submit `article-page.html`. Open it in a browser and confirm it looks like a real article page, not unstyled HTML.

# Assignment — What is Tailwind and Setup

## Goal

Build a styled "About Me" card page using only Tailwind utility classes — no custom CSS allowed. This proves you can set up Tailwind and use it to produce a real design.

## Requirements

- [ ] Set up Tailwind via the CDN `<script>` tag in the `<head>`
- [ ] Create a page with a centered card containing a name, role, short bio, and at least two skill tags
- [ ] Style the page background using a Tailwind background color (e.g., `bg-gray-100`)
- [ ] Style the card with a white background, rounded corners, padding, and a shadow (`bg-white rounded-xl p-8 shadow-md`)
- [ ] Use at least 8 different Tailwind utility classes across the page

## Deliverable

Submit `about-card.html`. The page must look styled — not plain HTML — when opened in a browser.

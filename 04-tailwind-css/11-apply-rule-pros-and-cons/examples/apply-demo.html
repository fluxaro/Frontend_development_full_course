@apply Demo

Before vs After — extracting repeated utilities into CSS classes

⚠️

CDN Limitation

@apply requires a PostCSS/Vite build step and does NOT work with the Tailwind CDN. This demo shows the concept using a <style> block with manual CSS values. In a real project, you would write .btn { @apply bg-blue-600 text-white ...; }

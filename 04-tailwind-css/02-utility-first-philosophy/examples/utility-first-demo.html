Utility-First Philosophy

One class = one CSS property. Compose them to build anything.

Spacing — p-*, m-*, gap-*

p-1 = 4px
p-2 = 8px
p-3 = 12px
p-4 = 16px
p-6 = 24px
p-8 = 32px

Scale: 1 unit = 4px. p-4 = padding: 1rem (16px)

bg-blue-500
bg-green-500

bg-red-500
bg-purple-500
bg-pink-500

leading-relaxed — comfortable for body text. This is a longer sentence to show the line height difference clearly.

leading-loose — very airy, good for short UI text

# Assignment — Responsive Photo Gallery with Grid

## What You Are Building

Create a responsive photo gallery using CSS Grid.

---

## Requirements

Create `photo-gallery.html` and `photo-gallery.css` with:

### Gallery Sections

1. **Featured row** — 3 columns: `2fr 1fr 1fr`
2. **Standard grid** — `repeat(auto-fill, minmax(250px, 1fr))`
3. **Masonry-style** — different row heights using `grid-template-rows`
4. **Magazine layout** — mixed column and row spans

### Technical Requirements

- Use `repeat()` for all regular grids
- Use `minmax()` for responsive columns
- Use `auto-fill` for the standard gallery
- Use `gap` for spacing
- Use `aspect-ratio` on images to maintain proportions

---

## Submission

Submit `photo-gallery.html` and `photo-gallery.css`.

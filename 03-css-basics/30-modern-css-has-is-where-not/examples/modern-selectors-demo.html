Modern CSS Selectors — :is(), :where(), :not(), :has()

:is() — Group Selectors

:is(h1, h2, h3) { color: blue } — all headings get the same style

This h3 is blue (styled by :is(h1, h2, h3))

This paragraph is styled by :is(article, section, aside) p

:where() — Zero Specificity (easy to override)

:where(header, footer) a { color: gray } — but a.override { color: red } wins
Normal link (gray from :where()) Override link (red from .override)

:not() — Exclusion

li:not(:last-child) { border-bottom } — all except the last
  • Item 1 (has border)
  • Item 2 (has border)
  • Item 3 (has border)
  • Item 4 (no border — it's the last)
button:not(:disabled):hover { opacity: 0.85 }

:has() — Parent Selector (the "if" selector)

.card:has(img) { padding: 0 } — card with image removes padding

Card with Image

This card has an image, so .card:has(img) removes the padding and the image fills the top.

Card without Image

This card has no image, so it keeps the default padding.

.field:has(input:invalid) label { color: red } — highlight label when input is invalid

Type something that's not an email to see the label turn red.

nav:has(.dropdown.open) { background: dark } — nav changes when dropdown is open
  • Home
  • About
  • Projects Blog Contact

Click "More" to open the dropdown — the nav background changes.

article:not(:has(img)) { max-width: 65ch } — articles without images are narrower

Article with Image

This article has an image, so it uses the full width.

Article without Image

This article has no image, so article:not(:has(img)) limits it to 65ch for better readability.

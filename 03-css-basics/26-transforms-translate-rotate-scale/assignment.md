# Assignment — Transform Effects Gallery

## What You Are Building

Create a gallery of UI effects using CSS transforms.

---

## Requirements

Create `transforms-gallery.html` and `transforms-gallery.css` with:

1. **Hover card flip** — card flips 180deg on hover (3D transform)
2. **Rotating loader** — spinning element using `rotate`
3. **Scale gallery** — images scale up on hover
4. **Skewed section** — section with skewed background
5. **Centered modal** — modal centered with `translate(-50%, -50%)`
6. **Accordion arrow** — arrow rotates 180deg when open
7. **Parallax effect** — elements at different `translateZ` depths

---

## Submission

Submit `transforms-gallery.html` and `transforms-gallery.css`.

# Assignment — Dashboard with Grid Template Areas

## What You Are Building

Build a complete admin dashboard layout using grid template areas.

---

## Requirements

Create `dashboard-grid.html` and `dashboard-grid.css` with:

### Desktop Layout (grid-template-areas)

```
"topbar  topbar  topbar"
"sidebar content widgets"
"sidebar footer  footer"
```

### Mobile Layout (single column)

```
"topbar"
"content"
"widgets"
"sidebar"
"footer"
```

### Each Area Must Contain

- **topbar** — logo, search bar, user avatar
- **sidebar** — navigation links with icons
- **content** — stats cards + data table
- **widgets** — quick stats, recent activity
- **footer** — copyright, links

---

## Submission

Submit `dashboard-grid.html` and `dashboard-grid.css`.

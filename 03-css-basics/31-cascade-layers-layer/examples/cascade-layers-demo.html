Unlayered styles always beat all layers, regardless of specificity.

Layer Order Declaration

/* Define order at the top — this is critical */
@layer reset, base, components, utilities;

/* Then fill each layer */
@layer reset    { /* CSS reset */ }
@layer base     { /* body, typography */ }
@layer components { /* buttons, cards */ }
@layer utilities  { /* .hidden, .text-red */ }

When to Use Cascade Layers

  • Managing third-party CSS (put it in a layer so your styles win)
  • Large codebases with multiple teams
  • Design systems with base + component + utility layers
  • Avoiding specificity wars

⚠️ Common Mistake

Not defining layer order at the top. If you don't declare the order, layers are ordered by first appearance — which can be unpredictable.

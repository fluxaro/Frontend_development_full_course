# Assignment — Polished Card Library

## What You Are Building

Create a complete card component library with 8 card variants.

---

## Requirements

Create `card-library.html` and `card-library.css` with these 8 card variants:

1. **Default** — subtle border, small shadow
2. **Elevated** — no border, large shadow
3. **Outlined** — colored border, no shadow
4. **Flat** — border only, no shadow
5. **Glass** — backdrop-filter blur effect
6. **Gradient border** — gradient border using `border-image` or pseudo-element
7. **Hover card** — shadow increases on hover with smooth transition
8. **Notification card** — left accent border, icon, title, message

Each card must have:
- A title
- A description paragraph
- A button or link
- Proper padding and border-radius

---

## Submission

Submit `card-library.html` and `card-library.css`.

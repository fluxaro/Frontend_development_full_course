# Assignment — Fully Accessible Animated Page

## What You Are Building

Create a page with animations that are fully accessible — respecting reduced motion preferences and providing excellent focus styles.

---

## Requirements

Create `accessible-animations.html` and `accessible-animations.css` with:

### Animations (with reduced motion fallbacks)

1. **Hero entrance** — slides up on load → fades in for reduced motion
2. **Card hover** — lifts with shadow → shadow only for reduced motion
3. **Button hover** — scale + shadow → color change only for reduced motion
4. **Loading spinner** — rotates → pulses for reduced motion
5. **Accordion** — height transition → instant for reduced motion

### Accessibility Requirements

- Skip link that appears on focus
- Visible focus styles on all interactive elements
- `prefers-reduced-motion` media query for all animations
- Sufficient color contrast (4.5:1 for normal text)
- No color-only information

---

## Submission

Submit `accessible-animations.html` and `accessible-animations.css`.

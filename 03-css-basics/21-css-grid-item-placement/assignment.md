# Assignment — Magazine Layout with Grid Placement

## What You Are Building

Create a magazine-style layout using grid item placement.

---

## Requirements

Create `magazine-layout.html` and `magazine-layout.css` with:

### Layout

A 12-column grid with these items:
- **Featured article** — spans columns 1–8, rows 1–3
- **Secondary article 1** — spans columns 9–12, rows 1–2
- **Secondary article 2** — spans columns 9–12, rows 3–4
- **Article row** — 4 equal articles spanning all 12 columns
- **Full-width banner** — spans all 12 columns

### Technical Requirements

- Use `grid-column: X / Y` for all placements
- Use `grid-row: X / Y` for multi-row items
- Use `grid-column: 1 / -1` for full-width items
- Use `span` keyword for at least 2 items

---

## Submission

Submit `magazine-layout.html` and `magazine-layout.css`.

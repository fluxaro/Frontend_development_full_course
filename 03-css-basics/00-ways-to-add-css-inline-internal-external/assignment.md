# Assignment — Style a Blog Post Three Ways

## What You Are Building

Style the same blog post HTML page using all three CSS methods, then refactor to use only external CSS.

---

## Requirements

### Part 1: Create the HTML

Create `blog-post.html` with a complete blog post structure:
- `<header>` with site name and nav
- `<main>` with an `<article>` containing:
  - `<h1>` article title
  - Author and date
  - At least 3 paragraphs
  - At least one `<h2>` section
  - At least one list
  - A `<blockquote>`
- `<aside>` with related links
- `<footer>`

### Part 2: Apply All Three Methods

1. **Inline CSS** — Apply at least 3 inline styles to specific elements
2. **Internal CSS** — Add a `<style>` block with at least 8 rules
3. **External CSS** — Create `blog-style.css` and link it with at least 10 rules

### Part 3: Refactor to External Only

Create a final version `blog-post-clean.html` that:
- Has NO inline styles
- Has NO `<style>` block
- Uses ONLY an external `blog-style.css`
- Looks identical to the version with all three methods

---

## What Good Looks Like

- The blog post is readable and well-styled
- The external CSS file is well-organized with comments
- The clean version looks the same as the mixed version
- The CSS uses meaningful property values (not just random colors)

---

## Submission

Submit all files: `blog-post.html`, `blog-post-clean.html`, and `blog-style.css`.

# Assignment — Complete UI with All Position Values

## What You Are Building

A complete page that uses all four position values in real-world contexts.

---

## Requirements

Create `positioning-demo.html` and `positioning-demo.css` with:

### Fixed Elements
- A sticky navigation bar that stays at the top when scrolling
- A "Back to top" button fixed to the bottom-right corner
- A notification toast fixed to the top-right corner

### Sticky Elements
- A sidebar that sticks while scrolling through content
- A table header that sticks when scrolling a long table

### Absolute Elements
- A card with a badge positioned at the top-right corner
- A dropdown menu that appears below a button
- An image with a caption overlay at the bottom

### Relative Elements
- A button that moves slightly on hover using `top` and `left`
- An underline effect using `::after` with `position: absolute`

---

## What Good Looks Like

- The navbar stays visible when scrolling
- The back-to-top button is always accessible
- Cards have properly positioned badges
- The page has enough content to scroll (at least 3 viewport heights)

---

## Submission

Submit `positioning-demo.html` and `positioning-demo.css`.

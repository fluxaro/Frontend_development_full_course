# Assignment — Profile Card with All Selector Types

## What You Are Building

Create a profile card component using all four CSS selector types.

---

## Requirements

### HTML Structure

Create `profile-card.html` with:
- A page with at least 3 profile cards
- Each card has: avatar (img), name (h3), role (p), bio (p), social links (a)
- One card should have a "featured" class in addition to the base "card" class
- The page has a header with an ID and a main section with an ID

### CSS Requirements

Create `profile-card.css` with:

**Universal selector:**
- CSS reset (box-sizing, margin, padding)

**Type selectors (at least 6):**
- body, h1, h2, h3, p, a, img

**Class selectors (at least 5):**
- `.card` — base card styles
- `.card.featured` — featured card variant
- `.avatar` — circular profile image
- `.card-name` — name styling
- `.social-link` — social link styling

**ID selectors (at least 2):**
- `#page-header` — page header
- `#cards-grid` — cards container

---

## What Good Looks Like

- The cards look like real profile cards
- The featured card is visually distinct from regular cards
- All selector types are used correctly
- The CSS is well-organized with comments separating sections

---

## Submission

Submit `profile-card.html` and `profile-card.css`.

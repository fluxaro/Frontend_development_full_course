Why Relative Units?

rem

Scales with the root font size. Respects user browser preferences. Use for font sizes and spacing.

clamp()

Fluid sizing between a minimum and maximum. Perfect for headings and padding that should scale with viewport.

dvh

Dynamic viewport height. Accounts for mobile browser UI. Use instead of vh for full-screen sections.

Optimal Reading Width

This text is constrained to 65ch — the optimal line length for reading. Studies show 45–75 characters per line is ideal for readability.

The font size uses clamp() to scale fluidly between 1rem on mobile and 1.125rem on large screens.

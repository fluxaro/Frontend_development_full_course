CSS Units Demo

px — Fixed pixels (not responsive)

width: 100px
width: 200px
width: 300px

% — Relative to parent width

width: 25%
width: 50%
width: 75%

rem — Relative to root font-size (16px default)

width: 1rem = 16px
width: 2rem = 32px
width: 4rem = 64px
width: 8rem = 128px
width: 16rem = 256px

em — Relative to parent font-size (parent = 20px here)

Parent font-size: 20px

width: 1em = 20px
width: 2em = 40px
width: 5em = 100px
width: 10em = 200px

vw — Relative to viewport width (resize the window!)

width: 25vw
width: 50vw
width: 75vw

ch — Width of "0" character (great for readable text)

max-width: 65ch — This box is limited to 65 characters wide, which is the ideal line length for comfortable reading. Notice how the text wraps naturally and stays easy to read without lines getting too long.

clamp(min, preferred, max) — Fluid sizing

font-size: clamp(1rem, 3vw, 2rem)
This text grows with the viewport but never goes below 1rem or above 2rem. Resize the window to see it change!

width: clamp(200px, 50%, 600px) — never smaller than 200px, never larger than 600px

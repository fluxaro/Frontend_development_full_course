# Assignment — Performance-Optimized Page

## What You Are Building

Create a performance-optimized page that demonstrates CSS performance best practices.

---

## Requirements

Create `performance-page.html` and `performance-page.css` with:

### Performance Techniques

1. **contain: content** on all card components
2. **will-change: transform** only on hover (not always)
3. **content-visibility: auto** on a long list (50+ items)
4. **contain-intrinsic-size** to prevent layout shift
5. All animations use only `transform` and `opacity`
6. No animations on `width`, `height`, `top`, `left`

### Page Content

- Navigation bar
- Hero section with entrance animation
- Card grid (12 cards) with hover effects
- Long list (50 items) with content-visibility
- Footer

---

## Submission

Submit `performance-page.html` and `performance-page.css`.

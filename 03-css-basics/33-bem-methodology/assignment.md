# Assignment — Build a UI Library with BEM

## What You Are Building

Create a complete UI component library using strict BEM naming.

---

## Requirements

Create `bem-library.html` and `bem-library.css` with these BEM components:

### Components

1. **Navigation** — `.nav`, `.nav__logo`, `.nav__list`, `.nav__item`, `.nav__link`, `.nav__link--active`
2. **Hero** — `.hero`, `.hero__content`, `.hero__title`, `.hero__subtitle`, `.hero__actions`
3. **Card** — `.card`, `.card--featured`, `.card__image`, `.card__body`, `.card__title`, `.card__text`, `.card__footer`
4. **Button** — `.btn`, `.btn--primary`, `.btn--secondary`, `.btn--danger`, `.btn--ghost`, `.btn--sm`, `.btn--lg`
5. **Form** — `.form`, `.form__group`, `.form__label`, `.form__input`, `.form__input--error`, `.form__hint`, `.form__error`
6. **Badge** — `.badge`, `.badge--success`, `.badge--warning`, `.badge--danger`, `.badge--info`
7. **Alert** — `.alert`, `.alert--success`, `.alert--warning`, `.alert--error`, `.alert__icon`, `.alert__message`

### BEM Rules

- No nested selectors (`.card .card__title` is forbidden)
- All selectors have specificity 0,1,0
- Modifiers always used with base class

---

## Submission

Submit `bem-library.html` and `bem-library.css`.

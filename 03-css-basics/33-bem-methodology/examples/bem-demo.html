BEM Methodology — Block__Element--Modifier

Naming Convention

.block — standalone component (.card, .nav, .btn)
.block__element — part of a block (.card__title, .nav__link)
.block--modifier — variant or state (.card--featured, .btn--primary)
.block__element--modifier — element variant (.card__btn--primary)

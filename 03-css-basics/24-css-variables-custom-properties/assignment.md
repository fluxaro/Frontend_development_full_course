# Assignment — Complete Design System with Themes

## What You Are Building

Create a complete design system with light and dark themes using CSS custom properties.

---

## Requirements

Create `design-system.html` and `design-system.css` with:

### Tokens (in `:root`)

- 5 color palettes (primary, secondary, success, warning, danger) with 9 shades each
- Typography scale (9 sizes)
- Spacing scale (12 steps)
- Border radius scale (5 values)
- Shadow scale (4 levels)

### Themes

- Light theme on `[data-theme="light"]`
- Dark theme on `[data-theme="dark"]`
- Theme toggle button using JavaScript

### Components (all using only `var()`)

- Navigation bar
- Hero section
- Card grid
- Form
- Buttons (5 variants)
- Badges

---

## Submission

Submit `design-system.html` and `design-system.css`.

Alerts — Scoped Variable Overrides

ℹ️ Info alert — uses --color-primary for the border
✅ Success alert — overrides --alert-border to --color-success
❌ Danger alert — overrides --alert-border to --color-danger
⚠️ Warning alert — overrides --alert-border to --color-warning

Fallback Values — var(--name, fallback)

When a variable is not defined, the fallback value is used:

var(--color-primary) — defined ✅ → blue
var(--undefined-color, #e74c3c) — undefined, fallback → red
var(--also-undefined, var(--color-success)) — chained fallback → green

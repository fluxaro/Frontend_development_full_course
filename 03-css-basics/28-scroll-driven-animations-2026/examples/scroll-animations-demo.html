Scroll Progress Bar
animation-timeline: scroll(root)

The blue bar at the top of the page

Uses animation-timeline: scroll(root) to track the page scroll position. The bar grows from 0% to 100% as you scroll from top to bottom. No JavaScript needed — it's driven entirely by CSS.

Quick Reference

scroll()

Tracks scroll position of a scrollable element. Use scroll(root) for the page.

view()

Tracks when an element enters/exits the viewport. Perfect for reveal animations.

animation-range

Controls when the animation plays: entry, exit, contain, cover.

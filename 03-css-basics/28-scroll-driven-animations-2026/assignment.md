# Assignment — Scroll-Animated Landing Page

## What You Are Building

Create a landing page where elements animate as you scroll.

---

## Requirements

Create `scroll-animations.html` and `scroll-animations.css` with:

1. **Reading progress bar** — fixed bar at top that fills as you scroll
2. **Fade-in sections** — each section fades in as it enters the viewport
3. **Staggered card reveal** — cards animate in one by one as you scroll
4. **Parallax header** — header image moves at different speed than content
5. **Counter animation** — numbers count up when they enter the viewport

Use `animation-timeline: scroll()` and `animation-timeline: view()`.

Include `@supports` fallback for older browsers.

---

## Submission

Submit `scroll-animations.html` and `scroll-animations.css`.

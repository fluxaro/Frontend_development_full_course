# Assignment — Style a Page Using Only Attribute Selectors

## What You Are Building

Style a complete page using ONLY attribute selectors — no class or ID selectors.

---

## Requirements

Create `attribute-selectors.html` and `attribute-selectors.css` with:

### HTML (no classes or IDs)

- Navigation with links (internal, external, mailto, tel)
- A form with various input types
- A list of links to different file types (PDF, ZIP, images)
- Cards using `data-*` attributes for variants

### CSS (attribute selectors only)

- Style external links differently from internal links
- Add icons to PDF, ZIP, and image links
- Style required form fields
- Style disabled form fields
- Use `data-variant` attribute for card styling
- Use `data-size` attribute for button sizing

---

## Submission

Submit `attribute-selectors.html` and `attribute-selectors.css`.

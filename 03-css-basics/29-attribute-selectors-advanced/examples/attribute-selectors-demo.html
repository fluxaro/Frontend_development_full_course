CSS Attribute Selectors

[attr] — Element has the attribute

[disabled] { opacity: 0.5; cursor: not-allowed; } | [required] { border-left: 3px solid orange; }

[attr="val"] — Exact value match

input[type="text"] { border: blue } | input[type="email"] { border: green } | input[type="password"] { border: red }

[attr^="val"] — Starts with value

a[href^="http"]::after { content: ' ↗' } | a[href^="mailto"]::before { content: '✉️' }
  • External link (starts with https)
  • External link (starts with http)
  • Email link (starts with mailto)
  • Phone link (starts with tel)
  • Internal link (no icon)

[attr$="val"] — Ends with value

a[href$=".pdf"] { color: red } | a[href$=".zip"]::before { content: '🗜️' }
  • Download PDF
  • Download ZIP
  • View PNG image
  • View SVG logo
  • Regular link (no icon)

[attr*="val"] — Contains substring

a[href*="github"] { color: #333; font-weight: bold }
  • GitHub repository
  • GitHub profile
  • Regular link

[attr~="val"] — Contains word (space-separated)

[class~="featured"] { gradient background }
This has class="card featured" — styled by [class~="featured"]
This has class="card" only — not styled

[attr|="val"] — Starts with val or val- (language codes)

[lang|="en"] | [lang|="fr"] | [lang|="ar"]

lang="en" — English text (normal style)

lang="en-US" — also matches [lang|="en"]

lang="fr" — Texte français (blue)

lang="ar" — نص عربي (RTL, purple, italic)

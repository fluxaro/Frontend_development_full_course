CSS Functions Demo

CSS Functions: calc(), min(), max(), clamp(), var()

calc() — Mix Different Units

width: calc(100% - 40px)
width: calc(50% + 20px)
width: calc(100% / 3)

⚠️ Spaces around + and - are required in calc()

min() — Use the Smallest Value

width: min(50%, 400px) — never wider than 400px

max() — Use the Largest Value

width: max(200px, 30%) — never narrower than 200px

clamp(min, preferred, max) — Fluid Sizing

font-size: clamp(1rem, 3vw, 2rem) — resize the window to see this text scale!

var() — Custom Properties with Fallback

color: var(--undefined-color, #333) — the fallback kicks in when the variable isn't defined

CSS Variables — Color Palette from :root

--color-primary
--color-secondary
--color-danger
--color-warning

Scoped Variable Override — Same Component, Different Theme

Default card — uses --card-bg: white
Danger card — overrides --card-bg and --card-border locally
Success card — overrides --card-bg and --card-border locally

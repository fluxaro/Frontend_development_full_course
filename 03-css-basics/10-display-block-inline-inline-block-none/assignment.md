# Assignment — Navigation and Badge System

## What You Are Building

Create a navigation system and badge component that demonstrates all display values.

---

## Requirements

### Navigation Bar
- Horizontal nav using `display: inline-block` on `<li>` elements
- Full-width click areas using `display: block` on `<a>` elements
- A mobile menu that uses `display: none` to hide/show

### Badge System
- Inline badges using `display: inline-block`
- Badges that appear next to text without breaking the line
- At least 5 badge variants (success, warning, danger, info, default)

### Show/Hide Demo
- A section that is hidden by default (`display: none`)
- A button that toggles it visible (use JavaScript or CSS `:target`)
- Demonstrate the difference between `display: none` and `visibility: hidden`

### Inline vs Block Demo
- Show the same element with `display: block`, `inline`, and `inline-block`
- Demonstrate that `width` works on block and inline-block but not inline

Submit `display-demo.html` and `display-demo.css`.

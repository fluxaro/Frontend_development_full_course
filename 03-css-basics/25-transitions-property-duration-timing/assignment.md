# Assignment — Interactive UI with Transitions

## What You Are Building

Create a UI with smooth transitions on all interactive elements.

---

## Requirements

Create `transitions-demo.html` and `transitions-demo.css` with:

### Elements with Transitions

1. **Navigation links** — color and underline transition on hover
2. **Buttons** — background, shadow, and transform on hover/active
3. **Cards** — lift effect (shadow + translateY) on hover
4. **Form inputs** — border color and shadow on focus
5. **Accordion** — height transition for expand/collapse
6. **Image overlay** — opacity transition on hover
7. **Toggle switch** — smooth sliding transition

### Technical Requirements

- Only animate `transform` and `opacity` for performance-critical animations
- Use `transition-delay` on at least one element
- Use `cubic-bezier` for at least one transition
- All transitions must respect `prefers-reduced-motion`

---

## Submission

Submit `transitions-demo.html` and `transitions-demo.css`.

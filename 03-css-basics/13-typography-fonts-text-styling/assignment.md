# Assignment — Typography System

## What You Are Building

Create a complete typography system for a blog, including a type scale, font pairing, and all text styling properties.

---

## Requirements

### Font Setup
- Load 2 Google Fonts: one for headings (serif or display), one for body (sans-serif)
- Define a complete type scale using CSS custom properties (xs through 5xl)
- Apply `font-display: swap` for performance

### Typography Demo Page
Create `typography-system.html` and `typography-system.css` with:

1. **Type Scale** — show all 9 sizes with labels
2. **Font Weights** — show 100–900 for your chosen font
3. **Heading Hierarchy** — h1 through h6 with proper sizing
4. **Body Text** — paragraph with optimal line-height and max-width
5. **Text Decorations** — underline, line-through, overline
6. **Text Transforms** — uppercase, lowercase, capitalize
7. **Truncation** — single-line and multi-line ellipsis
8. **Letter Spacing** — normal, tight, wide examples
9. **Blockquote** — styled with proper typography
10. **Code** — inline and block code styling

---

## Submission

Submit `typography-system.html` and `typography-system.css`.

CSS Typography

Font Families

Sans-serif: 'Inter' — Clean, modern, great for UI

Serif: 'Merriweather' — Classic, great for long-form reading

Monospace: 'Courier New' — Code, terminals, tabular data

Font Sizes (rem)

0.875rem — Small text, captions

1rem — Body text (default)

1.25rem — Large body / subheading

1.5rem — Heading level 3

2rem — Heading level 2

3rem — Hero heading

Font Weights

font-weight: 100 — Thin

font-weight: 400 — Regular (default)

font-weight: 600 — Semi-bold

font-weight: 700 — Bold

font-weight: 900 — Black

Line Height

line-height: 1.1 (tight)
Good for large headings. Too tight for body text — lines feel cramped and hard to read.
line-height: 1.5 (normal)
The sweet spot for body text. Comfortable reading experience without too much space between lines.
line-height: 2 (loose)
Very airy. Can work for short UI text but feels too spread out for paragraphs.

Letter Spacing

letter-spacing: -0.05em — Tight (good for large headings)

letter-spacing: 0 — Normal (default)

letter-spacing: 0.1em — Wide

letter-spacing: 0.2em + uppercase — Label style

Text Alignment

text-align: left — Default, best for most text
text-align: center — Good for headings and short text
text-align: right — Useful for numbers and RTL
text-align: justify — Stretches text to fill the line. Can create awkward spacing in narrow containers.

Text Decoration

text-decoration: none — removes underline from links

text-decoration: underline

text-decoration: line-through — strikethrough

text-decoration: overline

Text Transform

text-transform: uppercase — great for labels and nav

TEXT-TRANSFORM: LOWERCASE

text-transform: capitalize — every word starts with a capital

Text Truncation

Single-line truncation: This is a very long text that will be cut off with an ellipsis when it overflows the container.

Multi-line clamp (3 lines): This paragraph will be clamped to exactly three lines of text. Any content beyond the third line will be hidden and an ellipsis will appear at the end. This is very useful for card components where you want consistent heights.

Readable Article — max-width: 65ch

The Ideal Reading Width

Using max-width: 65ch limits the line length to approximately 65 characters, which research shows is the optimal range for comfortable reading. Lines that are too long make it hard for the eye to track back to the start of the next line.

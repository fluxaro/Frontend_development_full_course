# Assignment — Flexbox Layout System

## What You Are Building

Build a complete page layout using only flexbox container properties.

---

## Requirements

Create `flexbox-layout.html` and `flexbox-layout.css` with:

1. **Navigation bar** — logo left, links right, vertically centered
2. **Hero section** — content centered both horizontally and vertically
3. **Features row** — 3 feature cards evenly spaced
4. **Testimonials** — cards that wrap on smaller screens
5. **Footer** — 4 columns with space-between

All layouts must use ONLY flexbox container properties (no item properties yet).

---

## Submission

Submit `flexbox-layout.html` and `flexbox-layout.css`.

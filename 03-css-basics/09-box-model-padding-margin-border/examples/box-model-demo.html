Padding Shorthand

padding: 20px — all sides equal
padding: 10px 30px — top/bottom | left/right
padding: 5px 20px 15px — top | left/right | bottom
padding: 5px 20px 15px 40px — top | right | bottom | left

Centering with margin-inline: auto

max-width: 400px; margin-inline: auto — centered horizontally

Margin Collapse — Vertical margins collapse to the larger value

Box A — margin-bottom: 30px
Box B — margin-top: 20px

Gap between them = 30px (not 50px). The larger margin wins. This does NOT happen inside flex or grid containers.

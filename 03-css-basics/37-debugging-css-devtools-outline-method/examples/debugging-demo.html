Bug 2: Margin Collapse

❌ Bug — margins collapse

margin-bottom: 30px
margin-top: 20px

Gap = 30px (not 50px)

✅ Fix — use flex/grid or padding

padding-bottom: 30px
padding-top: 20px

Gap = 50px (no collapse)

Bug 3: z-index Not Working (missing position)

❌ Bug — no position on red box

z-index: 10
z-index: 20
(no position!)

Red box has z-index: 20 but no position — z-index is ignored

✅ Fix — add position: relative

z-index: 10
z-index: 20
position: relative

Red box now has position: relative — z-index works

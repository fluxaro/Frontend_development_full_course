Debugging Techniques

Outline Method

* { outline: 1px solid red; }

Reveals all element boxes without affecting layout.

Background Method

* { background: rgba(255,0,0,0.1); }

Shows element boundaries including padding.

DevTools

Right-click → Inspect → Styles panel shows all applied rules. Computed panel shows final values.

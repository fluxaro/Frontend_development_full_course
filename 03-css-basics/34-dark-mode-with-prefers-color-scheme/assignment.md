# Assignment — Complete Dark Mode Implementation

## What You Are Building

Create a complete website with both automatic (system) and manual dark mode.

---

## Requirements

Create `dark-mode-site.html` and `dark-mode-site.css` with:

### CSS Requirements

1. Define all colors as CSS custom properties in `:root`
2. Override for dark mode using `@media (prefers-color-scheme: dark)`
3. Override for manual toggle using `[data-theme="dark"]` and `[data-theme="light"]`
4. Smooth transition on all color changes

### JavaScript Requirements

1. Theme toggle button
2. Save preference to `localStorage`
3. Restore preference on page load
4. Detect system preference on first visit

### Components to Style

- Navigation bar
- Hero section
- Card grid (at least 3 cards)
- Form with inputs
- Footer

---

## Submission

Submit `dark-mode-site.html` and `dark-mode-site.css`.

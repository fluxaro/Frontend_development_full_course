# Assignment — Color Palette and Theme System

## What You Are Building

Create a complete color system for a design system using all CSS color formats.

---

## Requirements

### Part 1: Color Palette Page

Create `color-palette.html` and `color-palette.css` that displays:

1. **Primary palette** — 9 shades using HSL (100–900)
2. **Secondary palette** — 9 shades using HSL
3. **Neutral palette** — 9 grays using HSL
4. **Semantic colors** — success, warning, error, info using oklch
5. **Transparency demo** — show the same color at 10%, 25%, 50%, 75%, 100% opacity

Each color swatch must show:
- The color as a visual block
- The hex value
- The HSL value
- The oklch value

### Part 2: Theme Application

Apply your color system to a sample UI with:
- A navigation bar using primary colors
- Cards using neutral colors
- Buttons in primary, success, warning, and error variants
- A form with focus states using your primary color

### Part 3: Dark Mode Preview

Create a dark version of the same UI by changing only the CSS custom properties (variables).

---

## What Good Looks Like

- All 9 shades of each palette are visually distinct
- The semantic colors are accessible (sufficient contrast)
- The dark mode looks professional
- CSS custom properties are used for all colors

---

## Submission

Submit `color-palette.html` and `color-palette.css`.

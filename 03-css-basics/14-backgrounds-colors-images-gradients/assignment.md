# Assignment — Landing Page with Rich Backgrounds

## What You Are Building

Create a landing page that uses backgrounds creatively throughout.

---

## Requirements

Create `landing-backgrounds.html` and `landing-backgrounds.css` with:

### Sections Using Different Background Techniques

1. **Hero** — Full-screen with background image + gradient overlay + text
2. **Features** — Subtle gradient background
3. **Stats** — Dark background with CSS dot pattern
4. **Testimonials** — Light gradient background
5. **CTA** — Bold gradient background
6. **Footer** — Dark solid background

### Technical Requirements

- At least 3 different gradient types (linear, radial, conic)
- At least 1 CSS pattern using `repeating-linear-gradient`
- At least 1 gradient text effect
- Multiple backgrounds on at least 1 element
- `background-size: cover` on the hero image
- Fallback `background-color` for all image backgrounds

---

## Submission

Submit `landing-backgrounds.html` and `landing-backgrounds.css`.

# Assignment — Adaptive Card Component

## What You Are Building

Create a card component that adapts to its container using container queries.

---

## Requirements

Create `adaptive-card.html` and `adaptive-card.css` with:

### The Card Component

The same card HTML placed in 3 different containers:
1. **Narrow container** (250px) — stacked layout
2. **Medium container** (400px) — horizontal layout
3. **Wide container** (600px) — expanded layout with more details

### Container Query Breakpoints

```css
/* Narrow: default stacked */
/* Medium: @container (min-width: 350px) — horizontal */
/* Wide: @container (min-width: 550px) — expanded */
```

### Card Content

Each card must have: image, title, description, tags, and a button.

---

## Submission

Submit `adaptive-card.html` and `adaptive-card.css`.

Media Queries vs Container Queries

Media Query

Responds to the viewport size. Good for page-level layout changes. Not reusable — a card in a sidebar and a card in the main area get the same breakpoint.

Container Query

Responds to the container size. Perfect for reusable components. The same card adapts whether it's in a narrow sidebar or a wide main area.

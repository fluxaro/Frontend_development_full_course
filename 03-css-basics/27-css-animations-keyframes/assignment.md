# Assignment — Loading Screen and Animated UI

## What You Are Building

Create a loading screen and animated page entrance using CSS animations.

---

## Requirements

Create `animated-ui.html` and `animated-ui.css` with:

### Loading Screen
- Animated spinner using `rotate` keyframes
- Pulsing logo using `scale` keyframes
- Progress bar using `width` keyframes
- Fade out when loading completes

### Page Entrance Animations
- Hero heading slides up with fade
- Hero paragraph fades in with delay
- Cards stagger in (each with increasing delay)
- Navigation slides down from top

### Micro-interactions
- Button shake on error state
- Success checkmark draws itself
- Notification slides in from right

### Technical Requirements
- Use `animation-delay` for staggered effects
- Use `animation-fill-mode: forwards` to keep end states
- Use `animation-iteration-count: infinite` for looping animations
- All animations respect `prefers-reduced-motion`

---

## Submission

Submit `animated-ui.html` and `animated-ui.css`.

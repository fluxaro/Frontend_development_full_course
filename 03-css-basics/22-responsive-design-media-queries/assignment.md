# Assignment — Fully Responsive Landing Page

## What You Are Building

Create a fully responsive landing page that works on mobile, tablet, and desktop.

---

## Requirements

Create `responsive-landing.html` and `responsive-landing.css` with:

### Breakpoints

- Mobile: < 640px (default)
- Tablet: 640px–1023px
- Desktop: 1024px+

### Sections (all must be responsive)

1. **Navigation** — hamburger on mobile, horizontal on desktop
2. **Hero** — stacked on mobile, side-by-side on desktop
3. **Features** — 1 column → 2 columns → 3 columns
4. **Testimonials** — 1 column → 2 columns
5. **Footer** — stacked → 4 columns

### Technical Requirements

- Mobile-first approach (use `min-width`)
- At least 3 breakpoints
- `clamp()` for fluid typography
- `max-width: 100%; height: auto` on all images
- Navigation hides/shows with media query

---

## Submission

Submit `responsive-landing.html` and `responsive-landing.css`.

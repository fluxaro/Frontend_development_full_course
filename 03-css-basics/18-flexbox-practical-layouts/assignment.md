# Assignment — Complete Portfolio Page with Flexbox

## What You Are Building

Build a complete portfolio page using flexbox for every layout.

---

## Requirements

Create `portfolio-flexbox.html` and `portfolio-flexbox.css` with:

### Sections (all using flexbox)

1. **Navigation** — logo + links + CTA button
2. **Hero** — centered content, two CTA buttons side by side
3. **About** — image on left, text on right (media object pattern)
4. **Skills** — skill tags that wrap
5. **Projects** — 3-column card grid that wraps to 2 then 1 column
6. **Testimonials** — 2 testimonial cards side by side
7. **Contact** — form on left, contact info on right
8. **Footer** — 4 columns with links

### Technical Requirements

- Every layout uses flexbox
- No CSS Grid (save that for the next lesson)
- Cards use `flex: 1 1 280px` for responsive wrapping
- Sidebar/two-column layouts use `flex: 0 0 Xpx` and `flex: 1`
- Navigation uses `margin-left: auto` to push items to the right

---

## Submission

Submit `portfolio-flexbox.html` and `portfolio-flexbox.css`.

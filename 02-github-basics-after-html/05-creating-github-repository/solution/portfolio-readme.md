## 🛠️ Technologies Used

- **HTML5** — Semantic markup, accessibility (ARIA), forms
- **CSS3** — Flexbox, CSS Grid, custom properties, media queries
- **Git** — Version control with meaningful commit history
- **GitHub** — Remote repository and GitHub Pages hosting

## 🚀 How to View Locally

### Option 1: Clone and open

```bash
# Clone the repository
git clone https://github.com/yourusername/my-portfolio.git

# Navigate into the folder
cd my-portfolio

# Open in your browser
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

### Option 2: View live

Visit the GitHub Pages URL: `https://yourusername.github.io/my-portfolio`

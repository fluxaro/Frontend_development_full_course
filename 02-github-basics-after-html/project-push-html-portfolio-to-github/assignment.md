# Assignment — Complete HTML Portfolio on GitHub Pages

## What You Are Submitting

A complete, polished HTML portfolio website that is:
1. Hosted on GitHub in a public repository
2. Live on the internet via GitHub Pages
3. Built with everything you learned in the HTML module

---

## Portfolio Requirements

### index.html — Homepage
- Your name as `<h1>`
- Professional tagline
- Navigation to all pages
- A brief introduction section
- A "Featured Work" section with 1–2 project highlights
- Footer with copyright and links

### about.html — About Page
- Your story and background
- Why you are learning web development
- Skills section (what you know and what you are learning)
- Education or work history
- A photo or avatar (can be a placeholder)

### projects.html — Projects Page
- At least 3 projects from this bootcamp
- Each project has: title, description, technologies used, GitHub link
- Projects are in a list or grid layout

### contact.html — Contact Page
- A complete contact form (name, email, subject, message)
- Your email address as a `mailto:` link
- Your GitHub profile link
- Your LinkedIn profile link (or placeholder)
- Your location (city/country)

### README.md
- Project title
- Description of the portfolio
- Technologies used
- How to view it locally
- Live URL

### .gitignore
- Covers OS files, editor files, and any build output

---

## Git Requirements

- At least 10 commits
- Each commit has a specific, meaningful message
- Commits are incremental (not one big "add everything")
- `git log --oneline` tells the story of how the project was built

---

## GitHub Requirements

- Repository named `html-portfolio`
- Repository is public
- Has a proper README.md
- GitHub Pages is enabled and working

---

## What Good Looks Like

- The live URL works and loads quickly
- All navigation links work
- The contact form has proper labels and validation
- All images have alt text
- The page is accessible (keyboard navigable)
- The HTML is semantic and valid (passes W3C validator)
- The commit history is clean and meaningful

---

## Submission

Submit both:
1. GitHub repository URL: `https://github.com/yourusername/html-portfolio`
2. Live URL: `https://yourusername.github.io/html-portfolio`

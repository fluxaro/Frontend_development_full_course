git log & git diff

Inspect your commit history and see exactly what changed

git log — View Commit History

git log --oneline --graph --all
$ git log --oneline --graph --all
* a3f8d70 (HEAD → main) Add contact form validation
* b2c1e45 Style the navigation bar
* 9d4f123 Add about.html page
* 7e8a901 Initial commit: add index.html
git log --stat (shows which files changed)
$ git log --stat --oneline
a3f8d70 Add contact form validation
contact.html | 15 +++++++++++++++
style.css | 8 ++++++++
2 files changed, 23 insertions(+)

b2c1e45 Style the navigation bar
style.css | 24 ++++++++++++------------
1 file changed, 12 insertions(+), 12 deletions(-)
git log

Full log with author, date, and message

git log --oneline

Compact one-line format — hash + message

git log --oneline --graph --all

Visual branch graph — most useful format

git log -5

Show only the last 5 commits

git log --author="Alex"

Filter commits by author name

git log --since="1 week ago"

Filter commits by date range

git log --grep="fix"

Search commit messages for a keyword

git log -- index.html

Show commits that changed a specific file

git diff

Changes not yet staged (working directory)

git diff --staged

Changes staged but not yet committed

git diff HEAD~1

Compare last commit to current state

git diff abc1234 def5678

Compare two specific commits

git diff --name-only HEAD~1

Show only file names that changed

git diff HEAD~1 index.html

Diff a specific file only

Understanding HEAD

HEAD points to your current position in history:

7e8a901 Initial commit
9d4f123 Add about.html
b2c1e45 Style navigation
a3f8d70 Add form validation HEAD main

HEAD~1 = b2c1e45 (one before HEAD)
HEAD~2 = 9d4f123 (two before HEAD)
HEAD~3 = 7e8a901 (three before HEAD)

# Assignment — Text Processing Toolkit

## What You Are Building

An interactive text processing toolkit that applies string methods to user input in real time, demonstrating template literals and the full range of string methods.

---

## Requirements

Create `text-toolkit.html` with:

### Part 1 — Live String Inspector

A large textarea where the user types text. As they type, display in real time:
- Character count
- Word count
- Line count
- Unique word count
- Most common word
- Longest word
- Average word length

### Part 2 — String Transformations

Buttons that transform the text in the textarea:
- UPPERCASE
- lowercase
- Title Case (capitalise each word)
- Sentence case (capitalise first letter of each sentence)
- Reverse
- Remove extra spaces (replace multiple spaces with one)
- Slugify (for URL generation)

### Part 3 — Search and Replace

- A "Find" input and a "Replace with" input
- A "Replace All" button that uses `replaceAll`
- Highlight all occurrences of the search term in the text
- Show count of replacements made

### Part 4 — Template Literal Builder

A form with fields: name, role, company, years of experience.
Use a template literal to generate a professional bio paragraph that updates in real time as the user types.

### Part 5 — String Method Reference

A table showing every string method used in the toolkit with:
- Method name
- What it does
- Example input → output

---

## What Good Looks Like

- All transformations work correctly
- The live inspector updates on every keystroke
- Search and replace handles edge cases (empty search, no matches)
- The bio generator produces natural-sounding text

---

## Submission

Submit `text-toolkit.html`.

# Assignment — Calculator App

## What You Are Building

A functional calculator that uses all arithmetic operators, with a history log showing each operation.

---

## Requirements

Create `calculator.html` with:

### Calculator UI
- [ ] Display showing current value
- [ ] Buttons for digits 0–9
- [ ] Buttons for +, -, *, /, %, **
- [ ] Clear (C) and equals (=) buttons
- [ ] Decimal point button

### Operation History
- [ ] A list showing the last 10 operations
- [ ] Each entry shows: `expression = result`
- [ ] Colour-coded: positive results green, negative red

### Logical Operator Demo
- [ ] A section showing short-circuit evaluation
- [ ] Input for a username — if empty, show 'Anonymous'
- [ ] Input for an age — if under 18, show 'Minor', else 'Adult'

---

## Submission

Submit `calculator.html`.

# Assignment — Data Type Playground

## What You Are Building

An interactive HTML page that demonstrates all three primitive data types with live examples and a mini quiz that tests truthy/falsy knowledge.

---

## Requirements

Create `data-types-playground.html` with:

### String Section
- [ ] Display at least 8 different string methods with their output
- [ ] Show template literals with embedded expressions
- [ ] Show string concatenation vs template literals side by side

### Number Section
- [ ] Show integer, decimal, negative, and special values (Infinity, NaN)
- [ ] Demonstrate at least 6 Math methods with output
- [ ] Show the floating point precision issue (`0.1 + 0.2`)
- [ ] Show `toFixed()` as the solution for display

### Boolean Section
- [ ] List all 6 falsy values with a visual indicator
- [ ] Show 5 truthy values that might surprise beginners (`'0'`, `[]`, `{}`)
- [ ] Show comparison operators returning booleans

### Interactive Truthy/Falsy Checker
- [ ] An input field where the user types a value
- [ ] A button that evaluates whether the value is truthy or falsy
- [ ] Display the result with a green (truthy) or red (falsy) indicator

---

## What Good Looks Like

- Each section is clearly labelled
- Code examples show both the expression and the result
- The truthy/falsy checker works correctly for strings, numbers, and special values
- The page is styled and readable

---

## Submission

Submit `data-types-playground.html`.

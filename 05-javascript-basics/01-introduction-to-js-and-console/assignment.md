# Assignment — Console Explorer

## What You Are Building

A JavaScript file that uses every major console method and demonstrates basic JavaScript output. This proves you can set up JavaScript correctly and use the console as a debugging tool.

---

## Requirements

Create `console-explorer.html` with an embedded or linked `console-explorer.js` that:

### Console Output
- [ ] Uses `console.log` to output your name, age, and favourite language
- [ ] Uses `console.error` to log a fake error message
- [ ] Uses `console.warn` to log a fake warning
- [ ] Uses `console.table` to display an array of 3 objects (e.g., courses with name and duration)
- [ ] Uses `console.group` / `console.groupEnd` to group related logs
- [ ] Uses `console.time` / `console.timeEnd` to measure how long a loop takes

### Basic Calculations
- [ ] Logs the result of at least 5 arithmetic expressions
- [ ] Logs at least 3 comparison expressions (using `===`, `>`, `<`)
- [ ] Logs at least 2 string concatenations

### HTML Page
- [ ] The HTML page has a heading and a paragraph explaining what the script does
- [ ] Opening the page in a browser and checking the console shows all output

---

## What Good Looks Like

- The console shows organised, labelled output (not just random numbers)
- Groups are used to organise related output
- The timer shows a real measurement
- No errors in the console (only the intentional `console.error`)

---

## Submission

Submit `console-explorer.html` (and `console-explorer.js` if separate).

# Assignment — Internet Basics

## What You Are Building

Create an HTML page called `internet-journey.html` that explains how the internet works using only HTML structure — headings, paragraphs, lists, and tables.

No CSS. No JavaScript. Pure HTML only.

---

## Requirements

Your page must include all of the following:

### 1. A Main Heading
Use an `<h1>` with the title "How the Internet Works".

### 2. The Journey — 5 Steps as an Ordered List
Use an `<ol>` with 5 `<li>` items. Each item should be the name of a stop in the journey:
- Your Computer
- Your ISP
- DNS Server
- Web Server
- Response Back to You

Under each list item, add a short `<p>` explaining what happens at that step.

### 3. A Key Terms Table
Create a `<table>` with two columns: **Term** and **Definition**.
Include at least 6 terms from this lesson: Internet, Protocol, Packet, IP Address, Bandwidth, Latency, ISP, DNS, HTTP, HTTPS.

### 4. A Written Explanation Section
Add an `<h2>` called "How a Video Call Works" and write 3–5 sentences in your own words explaining what happens when you start a video call. Think about: packets, bandwidth, latency, and protocols.

### 5. An Analogy Section
Add an `<h2>` called "The Postal System Analogy" and use a `<table>` to compare internet concepts to postal system concepts (at least 5 rows).

---

## What Good Looks Like

- Every section has a clear heading
- The ordered list tells the story of a request step by step
- The table is properly structured with `<thead>` and `<tbody>`
- The written explanation is in your own words, not copied
- The page opens in a browser with no errors

---

## Submission

Save your file as `internet-journey.html` and submit it via GitHub or file upload.

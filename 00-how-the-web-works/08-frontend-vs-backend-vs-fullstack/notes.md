# Notes — Frontend vs Backend vs Fullstack

## Quick Comparison

| | Frontend | Backend | Fullstack |
|---|---|---|---|
| Runs in | Browser | Server | Both |
| Languages | HTML, CSS, JS | Python, Node, Go, etc. | Both |
| Frameworks | React, Vue, Angular | Express, Django, Rails | Both |
| Databases | No (reads from API) | Yes | Yes |
| Visible to users? | Yes | No | Both |

## Frontend Technologies

- **HTML** — Structure
- **CSS** — Styling
- **JavaScript** — Interactivity
- **React / Vue / Angular** — UI frameworks
- **Tailwind / Bootstrap** — CSS frameworks
- **Vite / Webpack** — Build tools

## Backend Technologies

- **Node.js** — JavaScript on the server
- **Python** — Django, Flask, FastAPI
- **Ruby** — Rails
- **PHP** — Laravel
- **Go** — Gin, Echo
- **PostgreSQL / MySQL** — Relational databases
- **MongoDB** — NoSQL database
- **Redis** — Caching

## Common Fullstack Stacks

| Stack | Technologies |
|---|---|
| MERN | MongoDB + Express + React + Node.js |
| MEAN | MongoDB + Express + Angular + Node.js |
| T3 | TypeScript + tRPC + Next.js + Prisma |
| LAMP | Linux + Apache + MySQL + PHP |
| JAMstack | JavaScript + APIs + Markup |

## Job Titles

**Frontend:** Frontend Developer, UI Developer, React Developer, Web Developer

**Backend:** Backend Developer, API Developer, Node.js Developer, Python Developer

**Fullstack:** Fullstack Developer, Software Engineer, Web Developer

## Key Insight

You do not need to know everything. Start with frontend. Get good at HTML, CSS, JavaScript, and React. Then learn backend when you are ready. Most companies hire specialists, not generalists.

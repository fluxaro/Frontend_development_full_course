# 08 — Frontend vs Backend vs Fullstack

## What Is This Lesson About?

Web development is split into different roles. Understanding what each role does helps you know what you are learning, what jobs exist, and how all the pieces fit together.

---

## Frontend Development

Frontend is everything the user sees and interacts with directly — the visual layer of a website.

**What frontend developers build:**
- The layout and design of web pages
- Navigation menus, buttons, forms
- Animations and transitions
- Responsive layouts that work on all screen sizes
- Interactive features (dropdowns, modals, sliders)

**Technologies:**
- HTML — structure
- CSS — styling
- JavaScript — interactivity
- React, Vue, Angular — UI frameworks
- Tailwind CSS, Bootstrap — CSS frameworks

**Analogy:** The dining room of a restaurant. It is what customers see and experience.

---

## Backend Development

Backend is everything that happens on the server — the logic, data, and security layer.

**What backend developers build:**
- APIs that the frontend calls to get data
- User authentication (login, signup, sessions)
- Database queries and data storage
- Business logic (pricing rules, permissions, workflows)
- Email sending, file uploads, payment processing

**Technologies:**
- Node.js, Python, Ruby, PHP, Java, Go — server languages
- Express, Django, Rails, Laravel — server frameworks
- PostgreSQL, MySQL, MongoDB — databases
- Redis — caching
- AWS, Google Cloud, Azure — cloud infrastructure

**Analogy:** The kitchen of a restaurant. Customers never see it, but it is where all the real work happens.

---

## Fullstack Development

A fullstack developer works on both frontend and backend. They can build a complete application from the database to the user interface.

**What fullstack developers build:**
- Complete web applications end-to-end
- APIs and the frontend that consumes them
- Database schemas and the UI that displays the data

**Common fullstack combinations:**
- MERN: MongoDB + Express + React + Node.js
- MEAN: MongoDB + Express + Angular + Node.js
- T3 Stack: TypeScript + tRPC + Next.js + Prisma
- LAMP: Linux + Apache + MySQL + PHP

**Analogy:** A chef who also manages the dining room and takes orders.

---

## Comparison Table

| Aspect | Frontend | Backend | Fullstack |
|---|---|---|---|
| What they build | UI, visuals, interactions | APIs, databases, server logic | Both |
| Languages | HTML, CSS, JS | Python, Node, Ruby, Go | Both |
| Frameworks | React, Vue, Angular | Express, Django, Rails | Both |
| Runs where? | In the browser | On the server | Both |
| Can users see the code? | Yes | No | Both |
| Entry-level salary (US) | $65k–$90k | $70k–$95k | $75k–$110k |

# Assignment — The URL Journey Page

## What You Are Building

Create an HTML page called `url-steps-page.html` that explains the 7 steps of what happens when you type a URL and press Enter.

No CSS. No JavaScript. Pure HTML structure only.

---

## Requirements

### 1. Main Heading
An `<h1>` with the title "What Happens When You Type a URL".

### 2. The 7 Steps as Sections
For each of the 7 steps, create a `<section>` containing:
- An `<h2>` with the step number and name (e.g., "Step 1: URL Parsing")
- A `<p>` explaining what happens in plain language
- Where relevant, a `<pre>` block showing an example (like an HTTP request or response)

The 7 steps are:
1. URL Parsing
2. DNS Resolution
3. TCP Three-Way Handshake
4. TLS Handshake (for HTTPS)
5. HTTP Request Sent
6. Server Sends HTTP Response
7. Browser Renders the Page

### 3. A Summary Table
After the 7 steps, add a `<table>` with three columns:
- Step Number
- What Happens
- Who Does It (Browser, DNS Server, Web Server, etc.)

### 4. A Key Terms Section
Add an `<h2>` called "Key Terms" and use a `<dl>` (description list) to define at least 6 terms from this lesson.

---

## What Good Looks Like

- Each step is clearly explained in your own words
- The `<pre>` blocks show realistic examples (HTTP request/response format)
- The summary table is complete and accurate
- The page is well-structured with proper heading hierarchy
- The page opens in a browser with no errors

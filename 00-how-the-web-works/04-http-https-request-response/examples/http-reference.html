HTTP Reference Guide

HTTP (HyperText Transfer Protocol) is the foundation of data communication on the web. This page covers the most important concepts.

HTTP Methods

HTTP methods tell the server what action to perform on a resource.

Common HTTP Methods
Method Purpose Has Body? Example Use
GET Retrieve data from the server No Load a webpage, fetch a list of users
POST Send data to create a new resource Yes Submit a form, create a new account
PUT Replace an entire resource Yes Update a user's full profile
PATCH Partially update a resource Yes Change only a user's email address
DELETE Remove a resource No Delete a post, remove an account
HEAD Like GET but returns headers only No Check if a resource exists
OPTIONS Ask what methods are allowed No CORS preflight requests

HTTP Status Codes

Every HTTP response includes a status code that tells you what happened.

2xx — Success

CodeNameMeaning
200OKRequest succeeded. Response body contains the result.
201CreatedNew resource was created successfully.
204No ContentSuccess but no response body (common after DELETE).

3xx — Redirection

CodeNameMeaning
301Moved PermanentlyResource has a new permanent URL.
302FoundTemporary redirect to another URL.
304Not ModifiedUse your cached version — nothing has changed.

4xx — Client Errors

CodeNameMeaning
400Bad RequestThe request was malformed or invalid.
401UnauthorizedAuthentication is required.
403ForbiddenAuthenticated but not allowed to access this.
404Not FoundThe resource does not exist at this URL.
429Too Many RequestsRate limit exceeded.

5xx — Server Errors

CodeNameMeaning
500Internal Server ErrorSomething went wrong on the server.
502Bad GatewayServer received an invalid response from upstream.
503Service UnavailableServer is down or overloaded.

HTTP vs HTTPS

HTTP vs HTTPS Comparison
Feature HTTP HTTPS
Encryption None — data is plain text TLS encryption — data is scrambled
Default Port 80 443
Security Anyone on the network can read data Data is private between browser and server
Use for passwords? Never Yes, safe
URL prefix http:// https://

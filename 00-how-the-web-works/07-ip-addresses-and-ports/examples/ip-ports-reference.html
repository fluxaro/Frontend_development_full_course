IP Addresses and Ports — Developer Reference

Every device on the internet has an IP address. Every service on that device listens on a port. Together they form the complete address of any service on the internet.

IPv4 vs IPv6

IPv4 vs IPv6 Comparison
Feature IPv4 IPv6
Format4 decimal numbers, dot-separated8 groups of 4 hex digits, colon-separated
Example192.168.1.12001:0db8:85a3::8a2e:0370:7334
Bit length32 bits128 bits
Total addresses~4.3 billion~340 undecillion
StatusStill dominantGrowing adoption

How to Find Your IP Address

Mac / Linux

ifconfig
# Look for "inet" followed by your IP address

# Or use the simpler command:
hostname -I

Windows

ipconfig
# Look for "IPv4 Address"

Find a Website's IP

nslookup google.com
nslookup github.com
ping github.com

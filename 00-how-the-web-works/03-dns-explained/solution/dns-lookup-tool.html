How to Run a Real DNS Lookup

Open your terminal and try these commands:

Using nslookup

nslookup github.com
nslookup google.com
nslookup -type=MX gmail.com

Using dig (Mac/Linux)

dig github.com
dig github.com A
dig github.com MX
dig github.com NS

DNS Record Types Reference

A Record
Maps a domain name to an IPv4 address. The most common record type.
AAAA Record
Maps a domain name to an IPv6 address.
CNAME Record
Creates an alias from one domain to another. Cannot be used on the root domain.
MX Record
Specifies the mail server responsible for accepting email for the domain.
TXT Record
Stores arbitrary text. Used for domain verification, SPF, DKIM, and DMARC email security.
NS Record
Lists the authoritative name servers for the domain.
TTL (Time To Live)
How many seconds a DNS answer can be cached before it must be looked up again.
